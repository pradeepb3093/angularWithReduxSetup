resolution modifier and their implementation

  • Optional

    This is type is added before declaring a dependency injector on a constructor to avoid getting run time error if the dependency is not provided in component or module or root level (the path in which the dependency resolution happens)

    constructor(@Optional() private serviceLoaclIdentifer: ServiceName)

    and Optional is imported from angular core

  • Self

    This is type is added before declaring a dependency injector on a constructor to create a local instance of the dependency inside the class its defined

    constructor(@Self() private serviceLoaclIdentifer: ServiceName)

    and Self is imported from angular core

  • SkipSelf

    This is type is added before declaring a dependency injector on a constructor to ignore local instance of the dependency inside the class its defined and consider the parent or the root instance

    constructor(@SkipSelf() private serviceLoaclIdentifer: ServiceName)

    and SkipSelf is imported from angular core

  • Host

    This is type is added before declaring a dependency injector on a constructor to take the object reference to its immediate parent if not will throw an error

    constructor(@Host() private serviceLoaclIdentifer: ServiceName)

    and Host is imported from angular core
